Responsive Bullet list built with the latest Bootstrap 5. Bullet lists are responsive components for displaying a series of content. Many examples and tutorials.

Check out [Bootstrap Bullet list Documentation](https://mdbootstrap.com/docs/standard/extended/bullet-list/) for detailed instructions & even more examples.

## Basic example
![image](https://user-images.githubusercontent.com/108793661/179505609-db60fd77-0a50-4d71-8a07-8a179e1f4bca.png)
```html
<ul>
  <li>Lorem ipsum</li>
  <li>Lorem ipsum</li>
  <li>Lorem ipsum</li>
  <li>Lorem ipsum</li>
  <li>Lorem ipsum</li>
</ul>
```

## How to use?

1. Download MDB 5 - free UI KIT

2. Choose your favourite customized component and click on the image

3. Copy & paste the code into your MDB project
